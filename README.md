# NextAI Agency

A multilingual AI consultancy website built with React, featuring:
- Multi-language support (EN, DE, ES, LV)
- Dark theme design
- Responsive layout
- Modern animations
- Detailed service pages

## Tech Stack

- React 18.2.0
- TypeScript
- Vite
- React Router
- i18next for translations
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Features

- Responsive navigation with mobile menu
- Language selector with automatic detection
- Service cards with detailed pages
- Dark mode optimized design
- Smooth transitions and animations

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/gatoraku/nextai-agency.git
cd nextai-agency
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
